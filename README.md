# README #

* 1. Start supervisor: /usr/local/bin/supervisord
* 2. Run supervisor CLI: http://localhost:9009
* 3. Run rabbitMQ: http://localhost:15672/#/
* 4. Run redis-cli and then MONITOR
* 5. Run scala worker `java jar worker.jar`
